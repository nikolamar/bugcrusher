------------

Client Js React is a UI library for creating bug reports. It can record the screen and all logs from the console. It enables you to push any data to report for analyzing and fixing. Thanks for using this library 🙌 and happy hunting 🎯.

<br />

Installation
------------

You can install using `npm`

```shell
npm install @bug-crusher/client-react--js
```

or `yarn`.

```shell
yarn add @bug-crusher/client-react-js
```
